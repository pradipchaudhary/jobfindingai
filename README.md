## Getting Started

### Prerequisites

- Node.js  
- Python 3.8+  
- MongoDB

### Installation

```bash
git clone https://github.com/jobfindingai/jobfindingai.git
cd jobfindingai
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```

**Backend:**
```bash
cd ../backend
pip install -r requirements.txt
python app.py
```

> Configure your environment variables in a `.env` file (Mongo URI, API keys, etc.)
